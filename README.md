# Анализ новостных текстов

Программа для автоматического сбора и семантического анализа новостей.

## Описание

Программа собирает новости из RSS-лент (РИА Новости, ТАСС, Коммерсантъ), выполняет лемматизацию и частотный анализ, строит облако слов и графики.

- Python (основной): парсинг, меню, вызов R
- R (вспомогательный): лемматизация, статистика, визуализация

## Установка

Требования:
- Python 3.8+
- R 4.0+

Python зависимости:
pip install -r requirements.txt

R пакеты:
install.packages(c("udpipe", "ggplot2", "dplyr", "wordcloud", "RColorBrewer"))

## Запуск

python start.py

## Меню программы

1. Парсинг новостей (Python)
   - Режим 1: собрать все новости
   - Режим 2: собрать за N дней (3, 7, 14)
   - Режим 3: добавить к существующим (накопление)

2. Запустить анализ (R)
   - Лемматизация текстов
   - Частотный анализ
   - Облако слов
   - Топ-15 слов
   - Динамика по дням

3. Открыть инструкцию (PDF)

0. Выход

## Результаты анализа

Файлы в папке output/:

- output/plots/wordcloud.png - облако слов
- output/plots/top15_words.png - топ-15 частотных слов
- output/plots/dynamics.png - динамика по дням
- output/tables/word_freq.csv - таблица частотности
- output/tables/dynamics.csv - таблица динамики
- output/report.txt - текстовый отчёт

## Возможные ошибки и решения

Ошибка: "FileNotFoundError: data/raw_news.csv"
Решение: сначала выполните парсинг новостей (пункт 1)

Ошибка: "Rscript не найден"
Решение: установите R и добавьте в PATH

Ошибка: "No module named 'feedparser'"
Решение: pip install feedparser

Ошибка: нет динамики по дням
Решение: нужно минимум 2 дня данных, собирайте новости несколько дней подряд в режиме 3
